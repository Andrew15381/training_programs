Скрипт для парсинга вакансий с hh.ru. Извлекает со страницы название, фирму, зарплату и навыки по части названия вакансии. Зарплата в рублях. Считает среднюют зп, медианную зп, разброс зп, 5 популярных навыков по части названия вакансии.
Загрузить вакансии в локальную БД:
hh --update <название вакансии>
Получить статистику из локальной БД:
hh --stats <название вакансии>
За особые старания по загрузке данных могут забанить по ip.
